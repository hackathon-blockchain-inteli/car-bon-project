# Carbon - Claim Offset Carbon just by driving 
Carbon is a platform that allows individuals to redeem Carbon Credits based on the measurement of the amount of CO2 emissions from their vehicles. This is done through an exchange for buying and selling these assets, using BTG dol. All of this is possible thanks to the combination of an IoT device, a web platform, and the infinite possibilities of Blockchain.

## Our solution
Carbon collects carbon emission data from cars using IoT, encouraging the use of electric or hybrid cars. The collected data is interpreted and can generate carbon credits on the blockchain, which can be sold on the platform to companies that want to offset their carbon footprint. The user has access to a dashboard that provides predictions generated by an AI regression model to visualize long-term impact and manage their credits. All transactions use BTG dol - a dollar-backed stablecoin - as currency, ensuring reliability, security, and transparency in the transactions carried out.
